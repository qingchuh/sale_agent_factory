# 🚀 前端快速启动指南

## 📋 前置要求

确保您的系统已安装：
- **Node.js** 16.0+ 
- **npm** 8.0+ 或 **yarn** 1.22+

## 🎯 一键启动（推荐）

### macOS/Linux 用户
```bash
./start_frontend.sh
```

### Windows 用户
```cmd
start_frontend.bat
```

## 🔧 手动启动

### 1. 进入前端目录
```bash
cd frontend
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问前端界面
打开浏览器访问: **http://localhost:3000**

## 🚨 常见问题

### Q: 依赖安装失败？
A: 尝试使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 端口被占用？
A: 修改 `vite.config.ts` 中的端口配置

### Q: 后端连接失败？
A: 确保后端服务在 http://localhost:8000 运行
